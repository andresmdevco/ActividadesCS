# 🧪 Automatización y Pruebas en Python

> Proyecto académico de pruebas y automatización en Python. Este repositorio contiene actividades prácticas relacionadas con:
> - Pruebas unitarias en Python.
> - Automatización web con Selenium.
> - Ejecución de pruebas en integración continua con GitHub Actions y Jenkins.

---

## ⚙️ Tecnologías utilizadas

- Python 3.x
- unittest
- Selenium WebDriver
- GitHub Actions (CI/CD)
- Jenkins

---

## 🚀 Cómo ejecutar el proyecto

### 1. Clonar el repositorio

```bash
git clone https://github.com/andresmdevco/ActividadesCS.git
cd ActividadesCS
```

### 2. Instalar dependencias

```bash
pip install selenium
```

> **Nota:** Asegúrate de tener configurado el WebDriver (por ejemplo, ChromeDriver)

---

## 🧪 Ejecutar pruebas

### Ejecutar un archivo específico:

```bash
python test1.py
```

---

## 🌐 Ejecutar scripts

### Ejemplos:

```bash
python extraerNumeros.py
python loginAutomatico.py
python navegarPestanas.py
```
